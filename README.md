# waterfalljs

![](http://files.cnblogs.com/files/MuYunyun/waterfall.gif)

[效果展示](muyunyun.cn/waterfall)

### feature

- [x] 支持同步/异步获取图片数据(基于发布/订阅模式)

- [x] 按需加载(每次渲染指定的条数)

- [x] 兼容各尺寸的屏幕

- [x] 无依赖(打包 JS 文件 3 kb)

- [x] 较好的扩展性

- [ ] 兼容性扩展

- [ ] UI 优化

### usage

```bash
npm install jswaterfall --save
```
